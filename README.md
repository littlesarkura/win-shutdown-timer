# win-shutdown-timer

一个轻量的 Windows 定时关机小工具，支持命令行版本和 GUI 图形界面版本。

## 功能

* 30 分钟后关机
* 1 小时后关机
* 2 小时后关机
* 自定义分钟或小时
* 取消定时关机
* 输入错误提示
* GUI 图形界面操作

## 运行环境

* Windows
* Python 3

## 项目结构

```text
win-shutdown-timer/
├─ .gitignore
├─ README.md
├─ main.py
└─ gui.py
```

## GUI 版本使用方法

运行：

```bash
python gui.py
```

运行后会打开一个图形界面窗口。

使用方式：

1. 在输入框中输入时间数字
2. 在下拉框中选择单位：分钟或小时
3. 点击“开始定时关机”
4. 如需取消，点击“取消定时关机”

示例：

```text
输入：30
单位：分钟
操作：点击开始定时关机
结果：设置 30 分钟后关机
```

## 命令行版本使用方法

运行：

```bash
python main.py
```

程序会显示菜单：

```text
Windows 定时关机小工具
1. 30 分钟后关机
2. 1 小时后关机
3. 2 小时后关机
4. 自定义时间
5. 取消定时关机
0. 退出程序
```

根据提示输入对应数字即可。

## 自定义时间

命令行版本中，选择 `4. 自定义时间` 后，需要输入时间数字和单位。

单位说明：

* `m`：分钟
* `h`：小时

示例：

```text
请输入时间数字：30
请输入单位，m 表示分钟，h 表示小时：m
```

表示 30 分钟后关机。

```text
请输入时间数字：2
请输入单位，m 表示分钟，h 表示小时：h
```

表示 2 小时后关机。

## 输入校验

如果输入了不支持的内容，程序不会直接异常退出，而是提示错误。

命令行版本示例：

```text
请选择功能：abc
功能选择错误，请重新输入
```

```text
请输入时间数字：abc
时间必须是正整数
```

GUI 版本中，输入错误时会弹出错误提示窗口。

## 原理

程序本身不会直接控制电脑电源，而是调用 Windows 自带的 `shutdown` 命令。

设置定时关机：

```bash
shutdown -s -t 3600
```

表示 3600 秒后关机。

取消定时关机：

```bash
shutdown -a
```
