# win-shutdown-timer

一个轻量的 Windows 定时关机命令行小工具。

## 功能

* 30 分钟后关机
* 1 小时后关机
* 2 小时后关机
* 自定义分钟或小时
* 取消定时关机
* 输入错误后返回菜单
* 支持退出程序

## 运行环境

* Windows
* Python 3

## 使用方法

克隆项目后，在项目目录运行：

```bash
python main.py
```

程序会显示菜单：

```text
Windows 定时关机小工具
1. 30 分钟后关机
2. 1 小时后关机
3. 2 小时后关机
4. 自定义时间
5. 取消定时关机
0. 退出程序
```

根据提示输入对应数字即可。

## 自定义时间

选择 `4. 自定义时间` 后，需要输入时间数字和单位。

单位说明：

* `m`：分钟
* `h`：小时

示例：

```text
请输入时间数字：30
请输入单位，m 表示分钟，h 表示小时：m
```

表示 30 分钟后关机。

```text
请输入时间数字：2
请输入单位，m 表示分钟，h 表示小时：h
```

表示 2 小时后关机。

## 输入校验

如果输入了不支持的内容，程序不会直接退出，而是提示错误后返回主菜单。

例如：

```text
请选择功能：abc
功能选择错误，请重新输入
```

或者：

```text
请输入时间数字：abc
时间必须是正整数
```

或者：

```text
请输入单位，m 表示分钟，h 表示小时：x
单位输入错误，只能输入 m 或 h
```

## 原理

程序本身不会直接控制电脑电源，而是调用 Windows 自带的 `shutdown` 命令。

设置定时关机：

```bash
shutdown -s -t 3600
```

表示 3600 秒后关机。

取消定时关机：

```bash
shutdown -a
```
